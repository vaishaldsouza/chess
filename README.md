### ⚙️ How It Works
When you select a move coordinate:
1. It opens a pre-filled GitHub Issue page (e.g., `Chess: Move E2 to E4`).
2. Simply click **"Submit new issue"** (do not edit the title or description).
3. A GitHub Actions workflow triggers, runs the underlying Python engine, updates this README, commits the changes, and closes the issue.
